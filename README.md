# Python for GIS

This repository is for the _Advanced Python Programming in GIS_ course for master students that I am teaching assistant for in Fall 2025 at the [University of Noshirvani, Babol](https://nit.ac.ir/en). The repository documents the journey through various Python projects, starting from fundamental programming concepts and progressing to geospatial applications and interactive web apps using Streamlit.
